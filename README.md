0:06:19 Setting up the development environment  

*run on Terminal:*  

```
node -v
```

```
npm i -g expo-cli
```

or  

```
sudo npm i -g expo-cli
```

[React Native Tools (Microsoft)](https://marketplace.visualstudio.com/items?itemName=msjsdiag.vscode-react-native)
